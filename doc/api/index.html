ELENA Object Library 1.5.4
API Specification

This document is the API specification for the ELENA Object Library, version 1.5.4.

Standard Package
std'basic Provides classes that describe the basic data types.
std'basic'math Provides classes that describe the basic mathematic operations.
std'basic'memory Provides classes that describe the basic operations with memory.
std'collections Provides classes that describe the basic operations with collections.
std'patterns Provides classes that describe the basic patterns.
std'properties Provides classes that describe the basic properties.


System Package
sys'events Provides the basic event handle routines.
sys'entries Provides the basic system entry symbols.
sys'culture Provides the system culture routines.

GUI Package
gui'controls Provides basic GUI controls.
gui'forms Provides basic GUI forms.
gui'graphics Provides basic graphic controls.

Win32 Package
win32'applications Provides classes that control the program flow.
win32'api Provides classes that contains the Win32 API wrappers.
win32'controls Provides the control wrappers
win32'api'factories Provides the factory classes
win32'api'graphics Provides the graphic wrappers
win32'api'styles Provides the handle styles.
win32'io Provides classes that describe the input/output operations.
win32'system Provides classes that implement OS-depended routines.
Win32'Socket Package
win32'socket'controls Provides classes that implements the socket input / output operations.
win32'api Provides socket api wrappers

Extended Package
ext'adapters Provides additional adapter classes.
ext'io Provides i/o helper classes.
ext'patterns Provides classes that describe the common patterns.
ext'text Provides classes implementing the patterns for working with text.
ext'utilities Provides utility classes.
Extended GUI Package
ext'gui'controls Provides gui control helpers.
