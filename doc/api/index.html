ELENA Object Library 2.0
API Specification

This document is the API specification for the ELENA Object Library, version 2.0.

System Package
system Provides classes that describe the basic functionality
system'calendar Provides basic operations with date / time
system'collections Provides classes that describe the basic operations with the collections
system'dynamic Contains classes supporting run-time code evaluation
system'io Provides basic i/o operations
system'math Provides basic mathematic extensions
system'routines Provides classes that describe the basic patterns
system'text Provides basic text operations

Extendion Package
extensions Contains the functionality extensions
extensions'math Contains the mathematical functionality extensions
