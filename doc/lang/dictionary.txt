Operations
==========
equal    : object   => bool
notequal : object   => bool
and      : object   => bool
or       : object   => bool
xor      : object   => bool
run      : action  
iterate  : function

Protocols
=========
action::
   eval   

bool_action::
   eval                           = bool

function::
   eval     : object

bool::
   then     : action              = object 
   then     : action &else:action = object
   else     : action              = object
   and      : bool/bool_action    = bool 
   or       : bool/bool_action    = bool 
   xor      : bool/bool_action    = bool 
   equal    : bool/bool_action    = bool
   notequal : bool/bool_action    = bool
   inverted                       = bool

interator::
   get                            = object
   next                           = bool

Subjects
========
if   :bool/bool_action
until:bool/bool_action
each :interator
